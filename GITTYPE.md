| 类型 | 说明 |
|------|------|
| `feature` | 新功能（feature） |
| `fix` | 修复 bug |
| `docs` | 文档更新 |
| `style` | 代码格式调整（不影响逻辑，如空格、分号） |
| `refactor` | 重构（既不是新功能也不是修复 bug） |
| `perf` | 性能优化 |
| `test` | 添加或修改测试 |
| `build` | 构建系统或依赖更新（如 webpack、npm） |
| `ci` | CI 配置或脚本变更 |
| `chore` | 杂项（如更新 .gitignore） |
| `revert` | 回滚之前的提交 |
