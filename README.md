# 💥libra-js-kit

一个专为各类业务系统打造的现代化 `JavaScript 工具包` ，助你写出更优雅、更规范、更可维护的代码

> 💻️代码也是我们程序猿的艺术品

- 🚀 基于 `TypeScript` 编写：类型安全，开发体验更佳，IDE 智能提示开箱即用
- 🧱 支持子路径导入`Tree-shakable`：按需引入，零冗余，极致优化打包体积
- 📦 覆盖常用场景：数组操作、日期处理、字符串工具、类型判断、通用工具函数等一应俱全
- ✅ 单元测试全覆盖：稳定可靠，放心集成到生产环境
- 📖 模块化设计：每个功能独立成模块，结构清晰，易于扩展与维护

为了让你写出更加优雅的代码
把那些细节交给工具包去处理，你可以专注于业务逻辑的实现

## 📦 安装

```bash
npm install libra-js-kit
```
或使用 Yarn / pnpm：
```bash
yarn add libra-js-kit
# 或
pnpm add libra-js-kit
```
